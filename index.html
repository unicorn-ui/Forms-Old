Getting Started

Setup Forms

  1. Download Forms and add files to your website.
  2. Add css in the head of your webpage. 
    <link rel="stylesheet" href="css/forms.css">
  3. Create forms in your html.

Customize Forms

  1. Fork or Download the Forms project on github.
  2. Make sure you have Sass installed.
  3. In the root of the Forms directory type: 
    sass --watch scss:css
    on the command line.
  4. Edit the _options.scss with your own custom values
  5. The css/forms.css file should now be updated
